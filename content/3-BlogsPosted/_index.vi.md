Tại đây sẽ là phần liệt kê, giới thiệu các blogs mà các bạn đã đăng trên [AWS Study Group](https://www.facebook.com/groups/awsstudygroupfcj). Ví dụ:

###  [Blog 1 - VPC ENDPOINT — CHÌA KHÓA ĐỂ XÂY SANDBOX CÔ LẬP TRÊN AWS](3.1-Blog1/)
Bài viết giới thiệu VPC Endpoint - giải pháp cho phép các tài nguyên bên trong VPC giao tiếp với dịch vụ AWS mà không cần đi qua Internet. Bao gồm hướng dẫn chi tiết cách xây dựng Sandbox cô lập, so sánh Gateway Endpoint và Interface Endpoint, cùng các lỗi thường gặp và cách khắc phục.

###  [Blog 2 - S3 PRESIGNED URL — UPLOAD FILE AN TOÀN MÀ KHÔNG CẦN ĐI QUA SERVER](3.2-Blog2/)
Bài viết hướng dẫn sử dụng Presigned URL để upload file lên S3 trực tiếp từ trình duyệt mà không cần qua Lambda, giải quyết bài toán giới hạn kích thước file 6MB của Lambda và 10MB của API Gateway.

###  [Blog 3 - AWS CONFIG AUTO-REMEDIATION — TỰ ĐỘNG KHÓA S3 BUCKET](3.3-Blog3/)
Bài viết giới thiệu cách sử dụng AWS Config kết hợp EventBridge và Lambda để tự động phát hiện và khóa lại S3 bucket ngay lập tức khi có ai đó lỡ tay bật Public Access, giúp ngăn ngừa rủi ro bảo mật do human error.
