This section will list and introduce the blogs you have posted to [AWS Study Group](https://www.facebook.com/groups/awsstudygroupfcj). For example:

###  [Blog 1 - VPC ENDPOINT — THE KEY TO BUILDING AN ISOLATED SANDBOX ON AWS](3.1-Blog1/)
This article introduces VPC Endpoint - a solution that allows resources within a VPC to communicate with AWS services without going through the Internet. Includes detailed instructions on how to build an isolated Sandbox, comparing Gateway Endpoint and Interface Endpoint, along with common errors and how to fix them.

###  [Blog 2 - S3 PRESIGNED URL — UPLOAD FILES SAFELY WITHOUT GOING THROUGH SERVER](3.2-Blog2/)
This article guides you through using Presigned URL to upload files to S3 directly from the browser without going through Lambda, solving the problem of Lambda's 6MB and API Gateway's 10MB file size limits.

###  [Blog 3 - AWS CONFIG AUTO-REMEDIATION — AUTOMATICALLY LOCK S3 BUCKET](3.3-Blog3/)
This article introduces how to use AWS Config combined with EventBridge and Lambda to automatically detect and lock S3 buckets immediately when someone accidentally enables Public Access, helping to prevent security risks caused by human error.
